# 🚀 Refresh Cafe - Setup

## Quick Start

```bash
# Install
npm install

# Run
npm run dev
```

## Firebase Setup

1. Go to [Firebase Console](https://console.firebase.google.com)
2. Deploy rules: `firebase deploy --only firestore:rules`
3. Enable Email/Password authentication
4. Create admin user: `admin@refresh-cafe.com` / `admin123`
